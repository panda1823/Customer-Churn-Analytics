# Customer Churn Analytics Dashboard

## Project Overview

This project analyzes telecom customer churn behavior to identify high-risk customer segments and key drivers influencing churn decisions. The goal is to support business teams in improving customer retention strategies through data-driven insights.

## Dataset

IBM Telco Customer Churn Dataset

Includes:

- Customer demographics
- Contract type
- Internet service
- Payment method
- Monthly charges
- Tenure
- Churn status

## Tools Used

Python
Power BI
Pandas
NumPy
Matplotlib
Seaborn

## Key Analysis Performed

- Data cleaning and preprocessing
- Handling missing values
- Feature-level churn comparison
- Tenure vs churn trend analysis
- Contract-based churn segmentation
- Monthly charges vs churn relationship analysis

## Dashboard KPIs

- Total Customers
- Churn Customers
- Retained customers
- Churn Rate
- Average Tenure

## Key Insights

- Month-to-month contract customers show highest churn risk
- Customers with higher monthly charges churn more frequently
- Fiber internet users show higher churn probability
- Customers with shorter tenure are more likely to leave

## Business Impact

This dashboard helps identify churn-prone customer segments and supports decision-making for retention strategies such as contract upgrades and pricing optimization.
